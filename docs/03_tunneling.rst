터널링의 물리적 배경
---------------------

고전 역학에서 입자의 에너지 :math:`E < V_0` 이면 포텐셜 장벽을 절대 넘을 수 없습니다.
그러나 양자역학에서는 파동함수가 장벽 안에서 **지수적으로 감소**하다가
반대편에서 0이 아닌 값을 가집니다. 이것이 **양자 터널링**입니다.

장벽 내부에서 슈뢰딩거 방정식:

.. math::

   -\frac{\hbar^2}{2m}\frac{d^2\psi}{dx^2} + V_0\psi = E\psi

:math:`E < V_0` 이면:

.. math::

   \frac{d^2\psi}{dx^2} = \kappa^2\psi, \quad \kappa = \sqrt{\frac{2m(V_0-E)}{\hbar^2}}

해: :math:`\psi(x) = Ae^{\kappa x} + Be^{-\kappa x}` → 진동이 아닌 **지수 감쇠**

그래프 1: ``03_rectangular_barrier.png``
-----------------------------------------

무엇을 보여주는가
~~~~~~~~~~~~~~~~~~

입자 에너지 :math:`E = 0.5, 1.0, 1.5` 에서 사각 장벽 (:math:`V_0 = 2.0, a = 2.0`) 통과:

- 왼쪽 영역: 입사파 + 반사파의 중첩 (진동하는 파동함수)
- 장벽 내부: 지수 감쇠
- 오른쪽 영역: 투과파 (작은 진폭)

투과 계수 계산 방법
~~~~~~~~~~~~~~~~~~~~

``03tunneling.py`` 는 **해석적 공식**과 **수치 계산** 두 가지를 모두 사용합니다.

해석적 투과 계수 (:math:`E < V_0`):

.. code-block:: python

   def calculate_transmission(E, V0, a):
       k = np.sqrt(2 * m * E / hbar**2)          # 장벽 밖 파수
       kappa = np.sqrt(2 * m * (V0 - E) / hbar**2) # 장벽 안 감쇠 계수
       T = 1 / (1 + (k**2 + kappa**2)**2 *
                np.sinh(kappa * a)**2 / (4 * k**2 * kappa**2))
       return T

수치 투과 계수는 파동함수의 진폭 비율로 계산:

.. math::

   T = \left|\frac{A_\text{transmitted}}{A_\text{incident}}\right|^2

왜 에너지가 높을수록 투과율이 높은가
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. math::

   T \approx e^{-2\kappa a}, \quad \kappa = \sqrt{\frac{2m(V_0 - E)}{\hbar^2}}

- :math:`E` 증가 → :math:`(V_0 - E)` 감소 → :math:`\kappa` 감소 → :math:`e^{-2\kappa a}` 증가
- :math:`E \to V_0` : :math:`\kappa \to 0` → 투과율 급증

실제 세계와의 대응
~~~~~~~~~~~~~~~~~~

- **STM (주사 터널링 현미경)**: 탐침과 표면 사이 진공 장벽을 전자가 터널링
  → 원자 단위 분해능으로 표면 이미징 가능
- **Flash 메모리**: 부유 게이트에서 전자가 산화막을 터널링 → 데이터 저장/삭제
- **알파 붕괴**: 알파 입자가 핵의 쿨롱 장벽을 터널링하여 방출

그래프 2: ``03_resonant_tunneling.png``
-----------------------------------------

무엇을 보여주는가
~~~~~~~~~~~~~~~~~~

두 개의 장벽으로 이루어진 이중 장벽 구조:

.. code-block:: text

   V(x): _____|‾‾|_____|‾‾|_____
             장벽1   우물  장벽2

투과율 대 에너지 그래프에서 **특정 에너지에서 T ≈ 1** 인 날카로운 피크가 나타납니다.

왜 특정 에너지에서 완전 투과인가
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

우물 안에 **준 속박 상태(quasi-bound state)** 가 형성됩니다.
입사 에너지가 이 준위와 일치하면:

1. 파동이 두 장벽 사이에서 공명 (standing wave)
2. 두 장벽에서의 반사파들이 **상쇄 간섭** (destructive interference)
3. 결과적으로 반사파 = 0, 투과파 = 1 (완전 투과!)

공명 에너지는 대략:

.. math::

   E_\text{res} \approx \frac{n^2\pi^2\hbar^2}{2mL^2}

(우물 너비 :math:`L` 의 무한 우물 에너지와 유사)

실제 세계와의 대응
~~~~~~~~~~~~~~~~~~

- **RTD (공명 터널링 다이오드)**: GaAs/AlAs 이중 장벽 구조, 수십 GHz 스위칭
- **양자 캐스케이드 레이저**: 공명 터널링으로 전자를 원하는 에너지 준위로 이동
- **부성 저항(Negative Resistance)**: 전압 증가 시 전류 감소 → 마이크로파 발진기

그래프 3: ``03_tunneling_parameters.png``
------------------------------------------

무엇을 보여주는가
~~~~~~~~~~~~~~~~~~

투과율 T를 세 가지 매개변수 각각의 함수로 보여줍니다:

1. **T vs 장벽 높이** :math:`V_0` (E, a 고정)
2. **T vs 장벽 너비** :math:`a` (E, V₀ 고정)
3. **T vs 입자 에너지** :math:`E` (V₀, a 고정)

왜 T가 장벽 두께에 지수적으로 감소하는가
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

:math:`T \approx e^{-2\kappa a}` 에서 :math:`a` 에 대한 지수 의존성이 직접 나타납니다.

STM의 원리도 이것입니다: 탐침과 표면 사이 거리 :math:`a` 가 0.1 Å 변하면
터널 전류가 약 2배 변합니다 — 원자 수준 분해능의 근거.

로그 스케일로 보면 직선: :math:`\ln T = -2\kappa a + \text{const}`

이것이 그래프에서 확인해야 할 핵심 특성입니다.
