그래프 1: ``02_gaussian_wave_packet.png``
------------------------------------------

무엇을 계산하는가
~~~~~~~~~~~~~~~~~~

가우시안 파동 패킷은 특정 위치 :math:`x_0` 주변에 국소화된 입자를 표현합니다:

.. math::

   \psi(x, 0) = A \exp\!\left(-\frac{(x-x_0)^2}{4\sigma^2}\right) \exp(ik_0 x)

파라미터:

- :math:`x_0 = 0` : 초기 위치
- :math:`\sigma = 2.0` : 위치 불확정성 (넓을수록 퍼진 파동)
- :math:`k_0 = 2.0` : 평균 운동량 (:math:`p_0 = \hbar k_0`)

코드에서의 구현:

.. code-block:: python

   psi_real = np.exp(-(x - x0)**2 / (4 * sigma**2)) * np.cos(k0 * x)
   psi_imag = np.exp(-(x - x0)**2 / (4 * sigma**2)) * np.sin(k0 * x)
   psi = psi_real + 1j * psi_imag

4개의 서브플롯이 각각 무엇인가
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

1. **Re[ψ(x)]** (파란색): 파동함수의 실수부 — 가우시안 포락선 안에서 :math:`\cos(k_0 x)` 진동
2. **Im[ψ(x)]** (빨간색): 파동함수의 허수부 — :math:`\sin(k_0 x)` 진동
3. **|ψ(x)|²** (녹색): 확률 밀도 — 순수한 가우시안 형태 (진동 없음!)
4. **arg(ψ)** (보라): 위상 — :math:`x_0` 근처에서 선형적으로 증가 (운동량 :math:`k_0` 반영)

왜 확률 밀도는 가우시안인가
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. math::

   |\psi(x)|^2 = |A|^2 \exp\!\left(-\frac{(x-x_0)^2}{2\sigma^2}\right)

:math:`\exp(ik_0 x)` 의 절댓값은 항상 1이므로 진동 성분이 사라지고
순수한 가우시안만 남습니다.

불확정성 원리 검증
~~~~~~~~~~~~~~~~~~~

이 파동 패킷에서:

.. math::

   \Delta x = \sigma = 2.0

.. math::

   \Delta p = \frac{\hbar}{2\sigma} = \frac{1}{4} = 0.25

.. math::

   \Delta x \cdot \Delta p = 2.0 \times 0.25 = 0.5 = \frac{\hbar}{2}

✅ **최소 불확정성 상태(Minimum Uncertainty State)** 를 구현합니다.
가우시안 파동 패킷이 바로 불확정성 원리의 등호 조건을 만족하는 유일한 함수입니다.

그래프 2: ``02_superposition_states.png``
------------------------------------------

무엇을 보여주는가
~~~~~~~~~~~~~~~~~~

무한 우물의 두 에너지 고유 상태를 중첩합니다:

.. math::

   \psi_\text{super} = \frac{1}{\sqrt{2}}(\psi_{n_1} + \psi_{n_2})

3가지 중첩 조합을 보여줍니다: (n=1 + n=3), (n=1 + n=5), 여러 상태 동등 중첩

왜 비대칭 분포가 나오는가
~~~~~~~~~~~~~~~~~~~~~~~~~~~

n=1과 n=3의 중첩:

- :math:`\psi_1(x) = \sqrt{2/L}\sin(\pi x/L)` : 한 봉우리
- :math:`\psi_3(x) = \sqrt{2/L}\sin(3\pi x/L)` : 세 봉우리
- 더할 때 **보강 간섭** 과 **상쇄 간섭** 이 발생 → 비대칭 확률 분포

.. math::

   |\psi_\text{super}|^2 = \frac{1}{2}(|\psi_1|^2 + |\psi_3|^2 + 2\psi_1\psi_3)

교차항 :math:`2\psi_1\psi_3` 가 간섭을 만듭니다.

실제 세계와의 대응
~~~~~~~~~~~~~~~~~~

- **슈뢰딩거의 고양이**: 중첩 상태는 측정 전까지 두 상태가 동시에 존재
- **큐비트**: :math:`|0\rangle + |1\rangle` 중첩이 양자 컴퓨터의 기본 연산
- **원자 간섭계**: 중성 원자의 중첩 상태를 이용한 초정밀 측정

그래프 3: ``02_hydrogen_orbitals.png``
----------------------------------------

무엇을 보여주는가
~~~~~~~~~~~~~~~~~~

수소 원자의 전자 오비탈을 1D 라디얼 파동함수로 시각화합니다.

포텐셜:

.. math::

   V(r) = -\frac{e^2}{4\pi\varepsilon_0 r} = -\frac{1}{r} \quad \text{(원자 단위)}

그래프는 :math:`r`에 따른 :math:`\psi(r)` 와 :math:`|\psi(r)|^2 \cdot r^2` (라디얼 확률 밀도)를 보여줍니다.

각 오비탈의 특성
~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :widths: 10 15 15 30

   * - 오비탈
     - 주양자수 n
     - 노드 수
     - 특징
   * - 1s
     - 1
     - 0
     - 원점에서 최대, 단조 감소
   * - 2s
     - 2
     - 1
     - 한 번 0 통과, 두 번째 봉우리 존재
   * - 2p
     - 2
     - 0 (라디얼)
     - 원점에서 0, 하나의 봉우리
   * - 3s
     - 3
     - 2
     - 두 번 0 통과

왜 1s가 원점에 최대인가
~~~~~~~~~~~~~~~~~~~~~~~~~

1s 파동함수: :math:`\psi_{1s}(r) = \frac{1}{\sqrt{\pi}a_0^{3/2}}e^{-r/a_0}`

원점 :math:`r=0` 에서 값이 최대입니다. 그러나 **라디얼 확률 밀도**:

.. math::

   P(r) = |\psi|^2 \cdot 4\pi r^2

는 :math:`r = a_0 = 0.529` Å (보어 반지름)에서 최대 — 전자를 가장 많이 발견하는 반지름.

실제 세계와의 대응
~~~~~~~~~~~~~~~~~~

- **화학 결합**: 오비탈이 겹치는 방식이 공유 결합의 강도와 방향 결정
- **원소 주기율표**: 전자 오비탈 채우기 (Aufbau principle)
- **분광학**: 전자 전이 시 방출/흡수하는 빛의 파장 예측
- **MRI**: 수소 핵의 스핀 상태 (오비탈의 자기 양자수와 관련)
