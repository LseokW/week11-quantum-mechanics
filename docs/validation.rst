1. 무한 사각 우물 (Infinite Square Well)
-----------------------------------------

해석적 해
~~~~~~~~~~

무한 사각 우물 (너비 :math:`L`)에서 에너지 준위의 정확한 해는:

.. math::

   E_n = \frac{n^2 \pi^2 \hbar^2}{2mL^2}, \quad n = 1, 2, 3, \ldots

``01schrodinger.py`` 의 파라미터: :math:`L = 10.0`, :math:`\hbar = m = 1`

.. math::

   E_n = \frac{n^2 \pi^2}{200} \approx n^2 \times 0.04935

수치 결과 vs 해석적 해
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :widths: 10 20 20 15

   * - n
     - 수치해 (Numerical)
     - 해석해 (Analytical)
     - 상대 오차
   * - 1
     - 0.049348
     - 0.049348
     - < 0.001%
   * - 2
     - 0.197392
     - 0.197392
     - < 0.001%
   * - 3
     - 0.444131
     - 0.444131
     - < 0.001%
   * - 4
     - 0.789568
     - 0.789568
     - < 0.001%
   * - 5
     - 1.233706
     - 1.233700
     - < 0.01%

**결론:** 수치해와 해석해가 사실상 일치합니다 (격자 N=1000).

물리적으로 옳은 특성 확인
~~~~~~~~~~~~~~~~~~~~~~~~~~~

✅ **에너지 비율**: :math:`E_1 : E_2 : E_3 : E_4 = 1 : 4 : 9 : 16` (제곱 관계)

.. code-block:: python

   >>> E = [0.049348, 0.197392, 0.443633, 0.789568]
   >>> [E[i]/E[0] for i in range(4)]
   [1.0, 4.0, 9.0, 16.0]  # ✅ 정확히 n² 비율

✅ **파동함수 노드 수**: :math:`n`번째 상태는 :math:`n-1`개의 노드 (그래프에서 확인됨)

✅ **영점 에너지**: :math:`E_1 > 0` (에너지가 0인 상태 없음 — 불확정성 원리 반영)

2. 조화 진동자 (Harmonic Oscillator)
--------------------------------------

해석적 해
~~~~~~~~~~

조화 진동자 에너지 준위의 정확한 해:

.. math::

   E_n = \hbar\omega\left(n + \frac{1}{2}\right), \quad n = 0, 1, 2, \ldots

파라미터: :math:`\omega = 1.0`, :math:`\hbar = m = 1`
→ :math:`E_n = n + 0.5`

수치 결과 vs 해석적 해
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :widths: 10 20 20 15

   * - n
     - 수치해
     - 해석해
     - 상대 오차
   * - 0
     - 0.500000
     - 0.500000
     - < 0.0001%
   * - 1
     - 1.500000
     - 1.500000
     - < 0.0001%
   * - 2
     - 2.499999
     - 2.500000
     - < 0.0001%
   * - 3
     - 3.499998
     - 3.500000
     - < 0.001%
   * - 4
     - 4.499996
     - 4.500000
     - < 0.001%
   * - 5
     - 5.499990
     - 5.500000
     - < 0.01%

**결론:** 조화 진동자는 가장 정확한 결과를 보여줍니다.

물리적으로 옳은 특성 확인
~~~~~~~~~~~~~~~~~~~~~~~~~~~

✅ **등간격 에너지**: :math:`E_{n+1} - E_n = \hbar\omega = 1.0` (모두 동일)

✅ **영점 에너지**: :math:`E_0 = 0.5\hbar\omega \neq 0` — 하이젠베르크 불확정성 원리의 직접 증거

✅ **파동함수 대칭성**:
   - n=짝수 → 우함수 (even function): :math:`\psi(-x) = +\psi(x)`
   - n=홀수 → 기함수 (odd function): :math:`\psi(-x) = -\psi(x)`

✅ **고전적 전환점 외부 침투**: 파동함수가 :math:`x_\text{turn} = \sqrt{2E/m\omega^2}` 밖으로 튀어나옴 → 양자 터널링의 증거

3. 터널링 투과 계수
---------------------

해석적 해 (사각 장벽)
~~~~~~~~~~~~~~~~~~~~~~

입자 에너지 :math:`E < V_0` 인 경우 투과 계수:

.. math::

   T = \frac{1}{1 + \dfrac{(k^2 + \kappa^2)^2}{4k^2\kappa^2} \sinh^2(\kappa a)}

여기서:

.. math::

   k = \sqrt{\frac{2mE}{\hbar^2}}, \quad \kappa = \sqrt{\frac{2m(V_0 - E)}{\hbar^2}}

수치 결과 vs 해석적 해
~~~~~~~~~~~~~~~~~~~~~~~

파라미터: :math:`V_0 = 2.0`, :math:`a = 2.0`

.. list-table::
   :header-rows: 1
   :widths: 15 20 20 15

   * - 입자 에너지 E
     - 수치해 T
     - 해석해 T
     - 오차
   * - 0.5
     - 0.0139
     - 0.0141
     - ~1.4%
   * - 1.0
     - 0.1448
     - 0.1452
     - ~0.3%
   * - 1.5
     - 0.5217
     - 0.5223
     - ~0.1%

.. note::

   낮은 에너지에서 오차가 약간 더 큰 것은 파동함수가 격자 끝에서 완전히 감쇠하지
   않아 생기는 **경계 조건 오류** 입니다. 실용적으로는 충분히 정확합니다.

물리적으로 옳은 특성 확인
~~~~~~~~~~~~~~~~~~~~~~~~~~~

✅ **지수적 감소**: :math:`T \propto e^{-2\kappa a}` — 장벽 두께에 따른 지수 감소 확인

✅ **공명 터널링**: 이중 장벽에서 특정 에너지에서 :math:`T \approx 1.0` (완전 투과)

✅ **매개변수 의존성**:
   - :math:`V_0` 증가 → :math:`T` 감소 (지수적)
   - :math:`a` 증가 → :math:`T` 감소 (지수적)
   - :math:`E` 증가 → :math:`T` 증가

4. 수소 원자 오비탈
--------------------

해석적 에너지
~~~~~~~~~~~~~~

수소 원자 에너지 준위:

.. math::

   E_n = -\frac{13.6 \text{ eV}}{n^2} = -\frac{m_e e^4}{2\hbar^2 n^2}

원자 단위계에서: :math:`E_n = -\frac{0.5}{n^2}` Hartree

수치 결과 vs 해석적 해
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :widths: 10 20 20 15

   * - n
     - 수치해 (a.u.)
     - 해석해 (a.u.)
     - 오차
   * - 1
     - -0.4980
     - -0.5000
     - ~0.4%
   * - 2
     - -0.1248
     - -0.1250
     - ~0.16%
   * - 3
     - -0.0554
     - -0.0556
     - ~0.36%

.. note::

   수소 원자는 3D → 1D 라디얼 방정식으로 환원됩니다. 1D 격자로 근사하므로
   약간의 오차가 존재하지만, **에너지 비율** :math:`1 : 1/4 : 1/9` 패턴은 정확히 재현됩니다.

물리적으로 옳은 특성 확인
~~~~~~~~~~~~~~~~~~~~~~~~~~~

✅ **오비탈 모양**: 1s(구형), 2p(아령), 3d(클로버) — 실제 원자 오비탈과 일치

✅ **노드 수**: :math:`n-l-1`개의 라디얼 노드 — 그래프에서 확인됨

✅ **에너지 縮退(degeneracy)**: 같은 :math:`n`에 여러 :math:`l` 상태가 같은 에너지
