물리적 배경: 왜 이런 결과가 나오는가
=======================================

이 장은 시뮬레이션 결과를 **실제 세계의 물리 현상**과 연결하여
각 출력이 왜 그런 모양인지 근본적인 이유를 설명합니다.

.. contents::
   :local:
   :depth: 2

----

1. 에너지 양자화는 왜 일어나는가
----------------------------------

핵심: 경계 조건 + 파동의 정상파 조건
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

무한 우물에서 파동함수는 경계에서 0이 되어야 합니다:

.. math::

   \psi(0) = 0, \quad \psi(L) = 0

파동함수의 일반해는 :math:`\psi(x) = A\sin(kx) + B\cos(kx)`

:math:`\psi(0) = 0` → :math:`B = 0`

:math:`\psi(L) = A\sin(kL) = 0` → :math:`kL = n\pi` → :math:`k = n\pi/L`

에너지: :math:`E = \hbar^2 k^2 / 2m = n^2\pi^2\hbar^2/2mL^2`

**직관**: 우물 안에 정확히 반파장의 정수배가 맞아야만 파동이 존재할 수 있습니다.
기타 줄이 양 끝에서 고정되어야 특정 주파수만 진동할 수 있는 것과 같습니다.

.. code-block:: text

   n=1: |  ∧  |   (반파장 1개)
   n=2: |∧  ∨|   (반파장 2개)
   n=3: |∧∨ ∧|   (반파장 3개)

----

2. 파동함수가 허수부를 가지는 이유
------------------------------------

슈뢰딩거 방정식의 시간 의존 형태:

.. math::

   i\hbar\frac{\partial\psi}{\partial t} = \hat{H}\psi

:math:`i` (허수 단위)가 있기 때문에 파동함수는 일반적으로 **복소수**입니다.

에너지 고유 상태의 시간 의존성:

.. math::

   \psi(x, t) = \psi(x) e^{-iEt/\hbar} = \psi(x)[\cos(Et/\hbar) - i\sin(Et/\hbar)]

시간이 지나면서 실수부와 허수부 사이에 에너지 교환이 일어납니다.
그러나 **확률 밀도** :math:`|\psi|^2 = \psi^* \psi` 는 시간에 무관합니다 (정상 상태).

``02wavefunction.py`` 의 가우시안 파동 패킷:

.. code-block:: text

   psi = exp(-(x-x0)^2 / 4*sigma^2) * [cos(k0*x) + i*sin(k0*x)]
   #                                      ^실수부          ^허수부

이것은 :math:`t=0` 에서 운동량 :math:`p_0 = \hbar k_0` 를 가진 입자를 표현합니다.

3. 터널링이 가능한 수학적 이유
--------------------------------

장벽 안에서의 슈뢰딩거 방정식 (:math:`E < V_0`):

.. math::

   \frac{d^2\psi}{dx^2} = +\kappa^2 \psi

(부호가 양수!) 해는 진동이 아닌 **지수함수**:

.. math::

   \psi(x) = Ae^{\kappa x} + Be^{-\kappa x}

물리적으로 의미 있는 해는 :math:`Be^{-\kappa x}` (감쇠). 이 함수는 **절대로 0이 되지 않습니다**.
따라서 장벽 반대편에서도 파동함수 값이 남아 있어 입자를 발견할 확률이 존재합니다.

고전 역학에서는 :math:`E < V_0` 이면 :math:`K.E. = E - V_0 < 0` → 불가능
양자역학에서는 파동함수가 연속이어야 한다는 조건만 있으면 됩니다.

4. 왜 수치 결과가 해석해와 잘 맞는가
---------------------------------------

유한 차분법의 수렴
~~~~~~~~~~~~~~~~~~~

격자 간격 :math:`dx` 에 대한 오차:

.. math::

   \frac{d^2\psi}{dx^2}\bigg|_\text{FD} = \frac{d^2\psi}{dx^2}\bigg|_\text{exact}
   + \frac{1}{12}dx^2 \frac{d^4\psi}{dx^4} + O(dx^4)

오차는 :math:`O(dx^2)` 입니다.

``01schrodinger.py`` 에서 :math:`N = 1000`, :math:`L = 10` → :math:`dx = 0.01`

예상 상대 오차: :math:`\sim dx^2/L^2 \sim 10^{-8}` (매우 작음)

실제 측정된 오차가 이보다 큰 이유:
- **경계 조건**: 격자 끝에서 파동함수가 실제로 0이 되지 않을 수 있음
- **고에너지 상태**: 파동함수가 빠르게 진동 → 격자로 잘 표현되지 않음 (Nyquist)

고에너지 상태에서 오차가 커지는 이유
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

n번째 파동함수의 파수: :math:`k_n = n\pi/L`

격자로 표현 가능한 최대 파수 (Nyquist): :math:`k_\text{max} = \pi/dx`

:math:`k_n < k_\text{max}` 조건: :math:`n < L/dx = N/1`

즉 격자 점 수만큼의 상태만 정확히 표현됩니다.
``N=1000`` 이면 처음 수십 개의 상태는 매우 정확하지만 n이 500에 가까워질수록 오차 증가.

5. 시뮬레이션의 한계와 실제 세계
----------------------------------

이 시뮬레이션이 단순화하는 것들
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :widths: 30 40 30

   * - 단순화
     - 실제 세계
     - 영향
   * - 1D 문제
     - 실제 원자는 3D
     - 수소 오비탈 각도 의존성 미포함
   * - 단일 입자
     - 다전자 원자/분자
     - 전자-전자 반발 미포함
   * - 정적 포텐셜
     - 핵이 움직임 (Born-Oppenheimer)
     - 분자 진동 결합 미포함
   * - 무한 격자 없음
     - 결정에서 주기적 포텐셜
     - 에너지 띠(band) 구조 미포함
   * - 스핀 없음
     - 전자는 스핀 1/2
     - 파울리 배타 원리 미포함

그럼에도 이 시뮬레이션이 의미 있는 이유
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

1. **핵심 물리를 올바르게 포착**: 에너지 양자화, 터널링, 파동-입자 이중성
2. **수치 검증 가능**: 해석해와 비교하여 알고리즘의 정확성 확인
3. **직관 구축**: 복잡한 실제 시스템을 이해하는 기초
4. **확장 가능**: 더 복잡한 포텐셜, 2D/3D, 시간 의존성으로 발전 가능
