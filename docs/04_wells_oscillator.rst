그래프 1: ``04_finite_well_depth.png``
----------------------------------------

무엇을 보여주는가
~~~~~~~~~~~~~~~~~~

우물 깊이 :math:`V_0 = 5, 10, 20` 에서 유한 우물의 에너지 준위와 파동함수:

왜 깊은 우물에 준위가 더 많은가
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

속박 상태의 조건: :math:`E < 0` (포텐셜이 :math:`-V_0` 이고 기준을 :math:`V=0` 으로 잡을 때)

속박 상태 수의 근사:

.. math::

   N_\text{bound} \approx \frac{a}{\pi\hbar}\sqrt{2mV_0}

- :math:`V_0` 가 2배 → 속박 상태 수 :math:`\sqrt{2} \approx 1.41` 배
- :math:`V_0 \to \infty` 일 때 속박 상태 수도 → ∞ (무한 우물로 수렴)

각 준위의 에너지가 왜 깊은 우물에서 낮아지는가
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

포텐셜이 깊어질수록 입자가 더 강하게 구속됩니다.
수학적으로: 같은 n의 준위라도 :math:`V_0` 가 커지면 절댓값 에너지가 커집니다 (음수이므로 더 낮아짐).

그래프 2: ``04_finite_well_width.png``
----------------------------------------

무엇을 보여주는가
~~~~~~~~~~~~~~~~~~

우물 너비 :math:`L = 2, 5, 10` 에서 에너지 준위 변화:

왜 좁은 우물에서 에너지가 높은가
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**하이젠베르크 불확정성 원리**의 직접적 결과:

.. math::

   \Delta x \cdot \Delta p \geq \frac{\hbar}{2}

우물 너비 = 위치 불확정성 :math:`\Delta x \sim L`

따라서:

.. math::

   \Delta p \gtrsim \frac{\hbar}{2L}

.. math::

   E \sim \frac{(\Delta p)^2}{2m} \gtrsim \frac{\hbar^2}{8mL^2}

:math:`L`이 절반이 되면 에너지가 4배 증가합니다.

.. note::

   이것이 나노 기술의 핵심 원리입니다. 양자점의 크기를 줄이면
   전자의 에너지 준위가 올라가 더 짧은 파장의 빛을 방출합니다.

실제 응용: 양자점 크기와 발광 색깔
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :widths: 20 20 20

   * - CdSe 양자점 크기
     - 에너지 갭
     - 발광 색깔
   * - ~2 nm
     - 높음
     - 파란색 (440 nm)
   * - ~4 nm
     - 중간
     - 초록색 (520 nm)
   * - ~7 nm
     - 낮음
     - 빨간색 (620 nm)

그래프 3: ``04_harmonic_oscillator_detailed.png``
--------------------------------------------------

무엇을 보여주는가
~~~~~~~~~~~~~~~~~~

n=0 ~ n=9 까지 조화 진동자 파동함수의 상세 분석:

- 파동함수 :math:`\psi_n(x)` 와 확률 밀도 :math:`|\psi_n(x)|^2`
- 고전적 전환점 표시: :math:`x_\text{turn} = \pm\sqrt{2E_n/(m\omega^2)}`
- 에너지 준위 등간격 확인

에르미트 다항식과의 연결
~~~~~~~~~~~~~~~~~~~~~~~~~

조화 진동자 파동함수의 정확한 형태:

.. math::

   \psi_n(x) \propto H_n\!\left(\sqrt{\frac{m\omega}{\hbar}}x\right) e^{-m\omega x^2/2\hbar}

에르미트 다항식 :math:`H_n`:

- :math:`H_0(u) = 1`
- :math:`H_1(u) = 2u`
- :math:`H_2(u) = 4u^2 - 2`
- :math:`H_3(u) = 8u^3 - 12u`

이를 통해 n=0은 순수 가우시안, n=1은 가우시안 × 홀함수, n=2는 2개 봉우리, ...

대응 원리 확인 (Correspondence Principle)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

n이 클수록 확률 밀도 :math:`|\psi_n|^2` 는 **고전적 확률 분포**에 가까워집니다.

고전적 조화 진동자의 확률 밀도:

.. math::

   P_\text{classical}(x) = \frac{1}{\pi\sqrt{A^2 - x^2}}

(진폭 :math:`A = x_\text{turn}`)

이것은 전환점 근처에서 높고 중심에서 낮은 형태입니다.
높은 n에서 양자 결과가 이 고전 분포에 수렴하는 것을 그래프에서 확인할 수 있습니다.

~~~~~~~~~~~~~~~~~~

세 가지 포텐셜의 에너지 준위를 한 그래프에 나란히 비교:

.. list-table::
   :header-rows: 1
   :widths: 30 25 25

그래프 5: ``04_classical_vs_quantum.png``
------------------------------------------

무엇을 보여주는가
~~~~~~~~~~~~~~~~~~

같은 에너지의 고전 입자와 양자 입자의 거동을 비교:

.. list-table::
   :header-rows: 1
   :widths: 30 30 30

   * - 특성
     - 고전 입자
     - 양자 입자
   * - 위치
     - 정확한 점
     - 확률 분포 :math:`|\psi|^2`
   * - 에너지
     - 연속 (임의)
     - 양자화 :math:`E_n`
   * - 최소 에너지
     - 0
     - :math:`E_0 = \frac{1}{2}\hbar\omega > 0`
   * - 전환점 밖
     - 절대 불가
     - 확률 있음 (터널링)

왜 영점 에너지가 0이 아닌가
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

만약 :math:`E = 0` 이면 입자는 :math:`x = 0` 에 정지 → :math:`\Delta x = 0, \Delta p = 0`

그러나 불확정성 원리: :math:`\Delta x \cdot \Delta p \geq \hbar/2`

따라서 :math:`E = 0` 은 불가능 → 최소 에너지 :math:`E_0 = \hbar\omega/2 > 0`

이것이 **영점 에너지(Zero-Point Energy)** 이며 우주 진공 에너지와도 연결됩니다.

실험적 증거
~~~~~~~~~~~

- **헬륨**: 0 K에서도 액체 상태 — 영점 에너지가 고체화를 방해
- **카시미르 효과**: 진공의 영점 에너지가 두 금속판을 끌어당김
- **분자 진동 스펙트럼**: 적외선 흡수 시 기저 상태에서도 진동 에너지 보유
